## Objetivo de la práctica

El objetivo de esta práctica es diseñar y desarrollar una web accesible e informativa sobre accesibilidad web, cumpliendo los requisitos de la normativa WCAG y la legislación española. La web presenta recursos, comparativas y buenas prácticas para concienciar sobre la importancia de la accesibilidad digital y facilitar la inclusión de todas las personas en el entorno online.

## Tecnologías usadas

- **HTML5**: para la estructura semántica.
- **CSS3**: hoja de estilos personalizada.
- **Bootstrap 5**: framework de componentes y grid responsive.
- **Bootstrap Icons**: iconos vectoriales integrados.
- **Fuentes locales (.ttf)**: uso de tipografías personalizadas.
- **JavaScript**: para funcionalidades interactivas como modals y tooltips.
